Extends the WooCommerce plugin with filters in administration.

== Description ==

It adds filters for products that give you more options:
- Filter for products by Shipping Classes

> This plugin requires the <a href="http://wordpress.org/extend/plugins/woocommerce/">WooCommerce plugin</a>

This plugin is maintained by [Closemarketing](https://www.closemarketing.es/)

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
